Sudoku Solver

Instructions

The game of Sudoku is played by placing numbers 1 through 9 into a grid of 9x9 squares. However, you have to ensure that each row and column only contains a single instance of each number from 1 to 9. Additionally, the 9 segments (separated by bold lines in the grid above) must also contain only one instance of each number. To play the game, select a number from the list below the Sudoku board and then click a cell in the board to populate it with the number you selected. This program will allow the user to play a game of Sudoku using the provided puzzle. When you believe you have the correct solution, you can click "Check Solution" to run the validating algorithm. A message will pop up on your screen if you have a correct solution or not. You can also click "AutoComplete" to see an example of a solved Sudoku game that passes the solution verification algorithm. Clicking "Reset Game" will reset the provided Sudoku puzzle, and clicking "Clear All" will completely clear the board and allow you to insert a different possible solution for verification.
